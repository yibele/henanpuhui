# 财务/管理层数据需求文档

> 本文档整理财务和管理层角色在系统中需要查看的数据模块

---

## 一、签约农户汇总

### 1.1 总体汇总数据

| 序号 | 数据项 | 单位 | 说明 |
|------|--------|------|------|
| ① | 已签约农户总数 | 户 | 所有状态为"已签约"的农户数量 |
| ② | 签约种植总面积 | 亩 | 所有签约农户的种植面积之和 |
| ③ | 定金总额 | 元 | 所有签约农户缴纳的定金总和 |
| ④ | 金/银/铜牌农户数量及占比 | 户/% | 按等级分类统计数量和占比 |

### 1.2 按负责人汇总数据

> 每个种植户负责人（业务员）的签约情况

| 序号 | 数据项 | 单位 | 说明 |
|------|--------|------|------|
| ⑤ | 所负责的签约农户数量 | 户 | 该负责人名下的签约农户数 |
| ⑥ | 所负责的签约种植面积 | 亩 | 该负责人名下农户的种植面积之和 |
| ⑦ | 所负责的农户缴纳定金总额 | 元 | 该负责人名下农户的定金总和 |
| ⑧ | 金/银/铜牌客户比例 | % | 该负责人名下农户的等级分布 |

---

## 二、种苗发放数据

### 2.1 昨日数据（每日更新）

| 序号 | 数据项 | 单位 | 说明 |
|------|--------|------|------|
| ① | 已发放苗的数量 | kg | 昨日发放种苗的总重量 |
| ② | 总价 | 元 | 昨日发放种苗的总金额 |
| ③ | 不同品种苗的发放数量 | kg | 按品种分类的发放数量明细 |
| ④ | 单价 | 元/kg | 各品种种苗的单价 |
| ⑤ | 已收款金额 | 元 | 昨日种苗款已收金额 |
| ⑥ | 欠款金额 | 元 | 昨日种苗款未收金额 |

### 2.2 截至昨日累计总数据

| 序号 | 数据项 | 单位 | 说明 |
|------|--------|------|------|
| ① | 发放苗的数量 | kg | 累计发放种苗的总重量 |
| ② | 总价 | 元 | 累计发放种苗的总金额 |
| ③ | 不同品种苗的发放数量 | kg | 按品种分类的累计发放数量 |
| ④ | 单价 | 元/kg | 各品种种苗的单价 |
| ⑤ | 已收款金额 | 元 | 累计种苗款已收金额 |
| ⑥ | 欠款金额 | 元 | 累计种苗款未收金额 |

### 2.3 发放进度

| 数据项 | 展示形式 | 说明 |
|--------|----------|------|
| 已签约农户苗发放占比 | 进度条 | 已发放种苗农户数 / 总签约农户数 |

---

## 三、公司农资使用数据

### 3.1 总体汇总

| 数据项 | 单位 | 说明 |
|--------|------|------|
| 使用公司农资的农户总数量 | 户 | 购买/使用公司农资的农户数 |
| 在签约种植农户中的占比 | % | 使用农资农户数 / 总签约农户数 |
| 农资销售金额 | 元 | 农资销售总金额 |

### 3.2 按负责人汇总

| 数据项 | 单位 | 说明 |
|--------|------|------|
| 使用公司农资的农户数量 | 户 | 该负责人名下使用农资的农户数 |
| 在其负责签约农户中的占比 | % | 占该负责人名下农户的比例 |
| 农资销售金额 | 元 | 该负责人名下农资销售金额 |

---

## 四、收苗数据

### 4.1 今日数据

#### 总体汇总

| 数据项 | 单位 | 说明 |
|--------|------|------|
| 收购总重量 | kg | 今日收购的总重量 |
| 总金额 | 元 | 今日收购的总金额 |
| 单价 | 元/kg | 总金额 / 总重量 |

#### 按仓库分类

| 数据项 | 单位 | 说明 |
|--------|------|------|
| 仓库收购总重量 | kg | 该仓库今日收购重量 |
| 仓库收购总金额 | 元 | 该仓库今日收购金额 |
| 仓库收购单价 | 元/kg | 该仓库今日平均单价 |

### 4.2 截至今日累计数据

#### 总体汇总

| 数据项 | 单位 | 说明 |
|--------|------|------|
| 收购总重量 | kg | 累计收购的总重量 |
| 总金额 | 元 | 累计收购的总金额 |
| 单价 | 元/kg | 累计平均单价 |
| 已卖叶子的农户数量 | 户 | 有收购记录的农户数 |

#### 按仓库分类

| 数据项 | 单位 | 说明 |
|--------|------|------|
| 仓库收购总重量 | kg | 该仓库累计收购重量 |
| 仓库收购总金额 | 元 | 该仓库累计收购金额 |
| 仓库收购单价 | 元/kg | 该仓库累计平均单价 |

---

## 五、结算数据

### 5.1 结算汇总

| 数据项 | 单位 | 说明 |
|--------|------|------|
| 已结算农户数量 | 户 | 完成结算的农户数 |
| 应付款总额 | 元 | 应支付给农户的总金额 |
| 已付款总额 | 元 | 已支付给农户的金额 |
| 待付款总额 | 元 | 尚未支付的金额（应付-已付） |

### 5.2 待结算列表

| 数据项 | 说明 |
|--------|------|
| 待结算农户清单 | 显示所有待结算的农户及其应付金额 |

## 六、页面模块划分

基于以上数据需求，财务/管理层首页建议划分为以下模块：

```
┌─────────────────────────────────────────────┐
│              核心指标概览卡片                  │
│  签约农户数 | 种植面积 | 定金总额 | 待结算金额   │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│              签约农户统计                      │
│  • 总体数据（农户数、面积、定金）               │
│  • 等级分布（金/银/铜牌饼图）                  │
│  • 按负责人分组统计表                         │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│              种苗发放统计                      │
│  • 昨日数据 / 累计数据（Tab切换）              │
│  • 发放进度条                                │
│  • 品种明细表                                │
│  • 收款/欠款情况                             │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│              收苗数据统计                      │
│  • 今日数据 / 累计数据（Tab切换）              │
│  • 按仓库分类统计                            │
│  • 已卖叶子农户数                            │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│              结算管理                         │
│  • 已结算/待结算（Tab切换）                   │
│  • 应付/已付/待付金额                         │
│  • 待结算农户列表                            │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│              公司农资统计                      │
│  • 总体使用情况                              │
│  • 按负责人统计                              │
└─────────────────────────────────────────────┘
```

---

## 七、数据更新频率

| 模块 | 更新频率 |
|------|----------|
| 签约农户汇总 | 实时 |
| 种苗发放数据 | 每日凌晨更新（昨日数据） |
| 收苗数据 | 实时 |
| 结算数据 | 实时 |
| 公司农资统计 | 实时 |

---

## 八、权限说明

- **财务/管理层**：可查看所有数据，可执行结算操作
- **业务员**：仅可查看自己负责的农户相关数据
- **仓库管理员**：仅可查看收苗和仓库相关数据
