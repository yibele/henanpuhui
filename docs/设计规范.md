# 普惠农务 - 设计规范文档

> 所有 CSS 变量已定义在 `app.wxss` 中，请直接使用。

## 一、字体大小规范

| 层级 | 变量名 | 大小 | 使用场景 |
|------|--------|------|----------|
| **3XL** | `--font-3xl` | 48rpx | 超大数字（emoji、图标） |
| **2XL** | `--font-2xl` | 40rpx | 特大标题 |
| **XL** | `--font-xl` | 36rpx | 页面标题、姓名 |
| **LG** | `--font-lg` | 32rpx | 金额数字、卡片标题 |
| **MD** | `--font-md` | 28rpx | **正文内容**（默认） |
| **SM** | `--font-sm` | 24rpx | 标签文字、辅助说明 |
| **XS** | `--font-xs` | 20rpx | 单位、时间戳、次要信息 |

### 使用规则

```css
/* ✅ 正确 - 使用变量 */
.title { font-size: var(--font-xl); }
.content { font-size: var(--font-md); }
.label { font-size: var(--font-sm); }

/* ❌ 错误 - 直接写数值 */
.title { font-size: 36rpx; }
```

### 场景对照表

| 场景 | 字体大小 | 字重 |
|------|----------|------|
| 页面标题 | `--font-xl` | 700 |
| 人名 | `--font-xl` | 700 |
| 区块标题 | `--font-md` | 700 |
| 金额数字 | `--font-lg` | 700 |
| 正文内容 | `--font-md` | 500 |
| 列表项标题 | `--font-md` | 600 |
| 输入框文字 | `--font-md` | 400 |
| 标签/说明 | `--font-sm` | 500 |
| 占位符 | `--font-md` | 400 |
| 单位/时间 | `--font-xs` | 400 |
| 徽章文字 | `--font-xs` | 600 |

## 七、快速参考

### 标题
```css
.page-title { 
  font-size: var(--font-xl); 
  font-weight: 700;
  color: var(--color-text-primary); 
}

.section-title { 
  font-size: var(--font-md); 
  font-weight: 700;
  color: var(--color-text-primary); 
}
```

### 正文
```css
.content { 
  font-size: var(--font-md); 
  font-weight: 500;
  color: var(--color-text-primary); 
}

.description { 
  font-size: var(--font-sm); 
  color: var(--color-text-secondary); 
}
```

### 数字
```css
.amount { 
  font-size: var(--font-lg); 
  font-weight: 700;
  color: var(--color-orange); 
}

.unit { 
  font-size: var(--font-xs); 
  color: var(--color-orange); 
}
```

### 卡片
```css
.card { 
  background: var(--color-bg-card);
  border-radius: var(--radius-lg);
  padding: var(--spacing-md);
  box-shadow: var(--shadow-sm);
}
```

## 八、工具类

全局样式已提供以下工具类，可直接使用：

```html
<!-- 字体 -->
<text class="text-xl font-bold">大标题</text>
<text class="text-md">正文</text>
<text class="text-sm text-secondary">说明</text>

<!-- 颜色 -->
<text class="text-primary">主色文字</text>
<text class="text-orange">橙色文字</text>

<!-- 间距 -->
<view class="p-md">内边距24rpx</view>
<view class="mt-md mb-sm">上边距24 下边距16</view>
<view class="gap-sm">子元素间距16rpx</view>

<!-- 布局 -->
<view class="flex items-center justify-between">...</view>

<!-- 圆角 -->
<view class="rounded-lg">圆角24rpx</view>
```
